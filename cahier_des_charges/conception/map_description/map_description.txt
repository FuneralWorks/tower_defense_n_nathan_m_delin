
Map.\textbf{create()}
Map.\textbf{getSelected()}
Map.\textbf{setSelected()}
Map.\textbf{play()}
Map.\textbf{show()}

Map.\textbf{create()}->liste de liste
        Description : crée la \emph{map} à partir d'un fichier .txt 
        stocker dans le module \textbf{Level} et à l'aide de la fonction split()
        Paramètres : \emph{fichier.txt}
        Valeur de retour : liste de liste

Map.\textbf{setSelected(m,i)}->rien
        Description : définit la case sélectionnée
        Paramètres : Map
                     index : tuple(entier,entier)
        Valeur de retour : rien

Map.\textbf{getSelected(m)}->tuple(entier,entier)
        Description : renvoie la case sélectionnée
        Paramètres : Map
        Valeur de retour : index de la case sélectionnée

Map.\textbf{play(g,p)}->int
        Description : pose une tour dans la case selectionnée
        Paramètres : Map
                     tower : caractère
        Valeur de retour : 1 si la case est vide, 0 si la case est occupée ou
        2 si la case est occupée par une tour qu'il est possible d'améliorer
Map.\textbf{show(m)}->rien
        Description : affiche la \emph{map}
        Paramètres : m: map
        Valeur de retour : rien
